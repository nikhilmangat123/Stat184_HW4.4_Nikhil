# Stat184_HW4.4_Nikhil

## About This Project

This repo contains a reproducible data analysis built in Quarto for STAT 184.
The analysis looks at passenger traffic trends across six major international
airports (2020-2024), beard length variation by college in the pirates dataset,
and a Monte Carlo simulation.

## Data Sources

- Airport traffic data: Wikipedia, List of Busiest Airports by Passenger Traffic
- Pirates dataset: {yarrr} R package
- Monte Carlo: simulated data generated in R
